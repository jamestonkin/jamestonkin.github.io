Trump and the Big Bubble

A single page application that tracks the tweets of the POTUS. The app makes API calls to retrieve the historical stock data of companies mentioned in the POTUS twitter feed. It then displays the stock prices in a graph 1 week before the tweet was made and 3 weeks after the tweet was made. This allows users to make correlations based on this data. Users can select companies to track and their settings are stored in a Firebase database. Technologies used: JavaScript, AngularJS, ChartJS, jQuery, Yahoo Query Language API.
